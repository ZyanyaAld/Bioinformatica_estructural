# Bioinformática Estructural

Repositorio del curso de Bioinformática Estructural (4to semestre,2026).

## Contenido

### Day 01 – Protein Structure and Folding

- **Estructura secundaria**: Conceptos fundamentales, hélices alfa, láminas beta y loops
- **Plegamiento y dinámica**: Factores que determinan el plegamiento proteico
- **Práctica con Foldit**: Introducción al plegamiento mediante gamificación

### Day 02 – Estructura Terciaria y Alineamiento Estructural

#### Teoría
- **Comparación de estructura terciaria**: La estructura está más conservada que la secuencia
- **Algoritmos de alineamiento estructural**: Foldseek, STAMP, SSAP, DALI, MAMMOTH, TM-align
- **Métricas de similitud**: RMSD, TM-score (>0.5 = mismo plegamiento), lDDT, E-value
- **Taxonomías estructurales**: CATH, SCOP y superfamilias

#### Práctica
- **Selección de dominios**: 7 estructuras de β-lactamasas desde CATH
  - Metaloproteasas (clase B): `3bc2A`, `3bv6A`, `3bk1A`, `3bk2A`
  - Serina proteasas (clase A): `3bebA`, `3becA`, `3blmA`
- **Alineamiento con FoldMason**: Superposición estructural múltiple
- **Búsqueda en FoldSeek**: Comparación contra base de datos PDB100
- **Resultados**: Identificación de homología estructural con RNasas J, confirmando que la estructura está más conservada que la secuencia
